# Wayne Drone Enterprises – Sales Operations Case Study

## Business Problem
Wayne Drone Enterprises relied on fragmented Excel files to manage accounts and opportunities, limiting reporting accuracy and CRM scalability.

## Project Goals
- Clean and standardize CRM data
- Prepare records for Salesforce import
- Build a structured sales pipeline
- Create executive-level dashboards

## Tools Used
- Excel
- Salesforce
- Power BI
- GitHub

## Key Outcomes
- Improved data consistency
- Standardized account structures
- Prepared CRM-ready datasets
- Created win/loss reporting dashboards

## Project Workflow
1. Raw Data Collection
2. Data Cleaning & Validation
3. CRM Preparation
4. Salesforce Import
5. Dashboard Development
6. Business Recommendations
